# Automated Tracer Diagnoses

Automated generation of a tracer diagnoses list from Alpha-ID-SE (ICD-10-GM / ORPHAcode connections).

## Download input files here:

- Alpha-ID-SE list from BfArM: https://www.bfarm.de/DE/Kodiersysteme/Services/Downloads/_node.html
- Complete list of ORPHAcodes from: https://www.orphadata.com
